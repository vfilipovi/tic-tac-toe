# Tic Tac Toe game

## Tech

- React, Typescript
- Tailwind CSS, NextUI
- Axios, React router, Jest

## Features

- Online multiplayer tic tac toe game
- Registration or login
- Paginated list of games
- Filter by status
- Create new game or join other games
